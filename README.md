# OMR and MP Editor Data
This branch contains the same final scores as the [`main` branch](https://github.com/martha-thomae/GuatC1), plus the output MEI files of the OMR and scoring-up processes, produced by MuRET and the Measurong Polyphony (MP) Editor, respectively. These files can be consulted in the `MuRET_and_MPEditor_files` folder that lies within each of the folders of the 27 pieces. The files ending in `_MURET.mei` are the ones generated by MuRET, and the ones ending in `_MENSURAL_PARTS.xml` and `_MENSURAL_SCORE.xml` are the ones generated by the MP Editor (the parts-based and the score-based MEI files, respetively).
